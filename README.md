## Build Instructions

Use CMake to build each example.

### Requirements
- CMake 3.10+
- C++17 or above
- A C++ compiler like g++ or clang++

### How to Build

```bash
cmake .
make
```

### Run

Navigate to the specific executable and run:

```bash
./01_print_hello_from_threads
```

## Cleaning the executables

```bash
make clean
```
